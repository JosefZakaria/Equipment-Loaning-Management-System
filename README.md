# Equipment-Loaning-Management-System
Simulates a real-time equipment loaning workflow using four concurrent threads (loan, return, admin, UI), with manager classes, randomized tasks, and shared resource handling.

**Equipment Loaning Management System**

A multithreaded Java application that simulates an equipment loaning system.
The program manages products, members, and loaned items while four concurrent threads handle loaning, returns, admin updates, and UI refreshing in real time. 


**Features**

Multithreading (4 parallel tasks)

Product, member, and loan item management

Randomized loan/return simulation

Clean separation of logic into manager classes

**How to run**

Open in any Java IDE and run the main class.
